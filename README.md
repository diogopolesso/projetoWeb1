##  Descrição do Projeto:

Este é um site completo em Atualização.
Site corporativo com foco em portfólio e serviços de tecnologia.

##  Seções incluídas:

- Hero — apresentação impactante com stats animados e grid geométrico de fundo
- Sobre — texto acolhedor com destaques visuais e badge de localização
- Serviços — 6 cards com animação de hover e tags de especialidade
- Skills — barras de progresso animadas por scroll + grid de tecnologias
- Portfólio — cards com overlay interativo e filtro por categoria
- Contato — formulário funcional com links diretos para LinkedIn, GitHub e e-mail
- Footer — links sociais e navegação
